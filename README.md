# 作品說明
面試作業 - 以元件化方式撰寫相關功能及css


## 系統說明
npm install, npm run dev




## 使用技術
* Vite
* React
* React Bootstrap
* SASS - 客製化scss變數
* Axios - 串接json資料(dummyjson API) 作為餐產品資訊
* React Router Dom - 頁面切換及商品資料頁動態更換
* Framer Motion - 過濾動畫


## 學習歷程

- 使用useParam 動態更換產品連結
